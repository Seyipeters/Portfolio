These steps require your signed-in LinkedIn session.

## A) Rebrand profile in 20 minutes
1. Update headline using `docs/linkedin_rebrand_kit.md`.
2. Replace About section with the provided summary.
3. Add featured links:
   - Portfolio: https://seyipeters.github.io/Portfolio/

4. Pin top skills: Python, SQL, Machine Learning, Data Analysis, scikit-learn, Git.

## B) Job search filters (Finland, English)

- Location: Finland
- Keywords:
  - `Machine Learning Intern`
  - `AI Intern`
  - `Data Science Intern`

- Language preference: English job description
